# Overview
This project aims to apply various machine learning algorithms to the data to predict a patient's dengue status based on clinical variables.

# Dengue
Dengue is a viral infection transmitted through mosquitoes, common in tropical and subtropical climates. Hundreds of millions of people are infected with the dengue virus every year, with roughly half the world's population at risk. While many cases of dengue are asymptomatic, some have symptoms such as high fever, headaches, muscle and joint pains, nausea, vomiting, and rash. Rare cases can lead to death.

# Dataset
This data contains patient-level clinical data critical for identifying dengue infection of eleven variables: age, gender, hemoglobin level, white blood cell (WBC) count, differential WBC count, red blood cell (RBC) count, platelet count, platelet size variation, and a dengue label. 

Note: This is a synthetic dataset, not from real patients, intended for educational use only.

# Methods
Machine learning methods for this project include
- Logistic Regression
- Random Forest 
- XGBoost
- K-nearest neighbors
